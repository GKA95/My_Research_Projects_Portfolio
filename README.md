# [Project 1: Household Power Consumption Forecasting](https://github.com/GKA95/Household-Power-Consumption-Forecasting)


### Overview

Developed an end-to-end machine learning solution to forecast household electricity consumption using historical energy usage data. The project focused on identifying consumption patterns, engineering predictive features, and evaluating multiple forecasting models to improve the accuracy of residential energy demand predictions.

### Problem Statement

Accurate forecasting of household electricity demand is essential for energy planning, smart grid operations, and efficient resource allocation. Traditional forecasting methods often struggle to capture the complex and dynamic nature of residential energy consumption. This project addresses that challenge by leveraging machine learning techniques to predict future power usage based on historical consumption patterns.

### Key Contributions

* Collected and preprocessed household energy consumption data.
* Performed exploratory data analysis to uncover usage trends and seasonal patterns.
* Engineered time-based and consumption-related features to enhance predictive performance.
* Developed and compared multiple machine learning forecasting models.
* Evaluated model performance using industry-standard metrics.
* Generated actionable insights to support energy management and demand forecasting.

### Technologies Used

Python • Pandas • NumPy • Scikit-Learn • Matplotlib • Seaborn • Jupyter Notebook

### Results

* Identified significant daily, weekly, and seasonal consumption patterns.
* Improved forecasting performance through feature engineering and model optimization.
* Demonstrated the effectiveness of machine learning for residential load forecasting.
* Produced insights that can support smart grid planning and demand-side energy management.

### Skills Demonstrated

Machine Learning • Time-Series Forecasting • Data Analytics • Energy Analytics • Data Visualization • Feature Engineering • Statistical Analysis • Predictive Modeling

### Impact

This project demonstrates how data-driven forecasting techniques can improve energy planning, support smarter electricity distribution, and contribute to more efficient and sustainable power systems.


# [Project 2: Autonomous Leader-Follower Vehicle Simulation Using CARLA and the Intelligent Driver Model (IDM)](https://drive.google.com/drive/u/0/folders/1bAUu1WD1lsh--vFAfmNk5Z3NFscf8uag)

### Overview

Developed an autonomous leader-follower vehicle simulation in CARLA 0.9.10 using the Intelligent Driver Model (IDM) to replicate realistic car-following behavior in a virtual driving environment. The project evaluated how an autonomous follower vehicle responds to changing traffic conditions while maintaining safe following distances, smooth acceleration, and collision-free operation.

### Problem Statement

Autonomous vehicles must continuously adapt their speed and spacing relative to surrounding traffic while ensuring safety and passenger comfort. This project investigated how the Intelligent Driver Model (IDM) can be used to achieve realistic and stable vehicle-following behavior in a high-fidelity simulation environment.

### Key Contributions
Implemented the Intelligent Driver Model (IDM) from mathematical theory into Python.
Developed a leader-follower vehicle control system using the CARLA autonomous driving simulator.
Simulated real-world traffic scenarios including cruising, controlled deceleration, and complete stopping.
Recorded and analyzed vehicle position, speed, acceleration, and inter-vehicle gap data.
Generated trajectory, speed, acceleration, and space-time visualizations for performance evaluation.
Validated safe gap maintenance and collision avoidance under dynamic driving conditions.
Technologies Used

Python • CARLA Simulator • OpenCV • Matplotlib • NumPy • Traffic Flow Modeling • Autonomous Systems

### Simulation Scenario

The simulation consisted of three traffic phases:

Cruising Phase – Leader vehicle maintained a constant speed while the follower stabilized behind it.
Deceleration Phase – Leader vehicle gradually reduced speed to a complete stop, requiring adaptive braking by the follower.
Stationary Phase – Both vehicles remained stopped while preserving a safe following distance.
Results
Successfully maintained safe following distances throughout the simulation.
Achieved smooth acceleration and braking without abrupt control actions.
Prevented collisions under all tested driving conditions.
Demonstrated stable vehicle behavior during cruising, braking, and stationary scenarios.
Produced repeatable results using synchronous simulation mode.
Skills Demonstrated

Autonomous Driving • Vehicle Control Systems • Traffic Flow Modeling • Python Programming • Simulation Engineering • Data Analysis • Algorithm Development • System Validation

### Impact

This project demonstrates the application of intelligent vehicle control algorithms in autonomous transportation systems. It highlights the ability to translate theoretical traffic models into practical simulation environments and evaluate their effectiveness through data-driven analysis.
